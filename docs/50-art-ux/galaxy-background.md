# Galaxy Background — 정적 캐주얼 우주 배경 & 루핑 반짝임

> 문서 범위 ④-b — 보드와 Title이 공유하는 우주 배경의 책임 분리를 못 박는다. 보드 내부 레이어는
> [[screen-structure]], Title 배경은 [[title-screen]]이 정본이다.

## 1. 책임 분리

Galaxy background는 **정적 캐주얼 우주 배경 레이어**와 **런타임 반짝임 효과 레이어**의 조합이다.

| 레이어    | 담당                               | 적용 화면                           |
| ------ | -------------------------------- | ------------------------------- |
| 정적 이미지 | 어두운 우주 톤, 플랫한 세로 물결형 은하 밴드, 낮은 대비의 큰 면 구성 | Pool In-Game 보드 내부, Title 전체 배경 |
| 런타임 효과 | 작은 별, 점광, 미세 입자, 알파/스케일 루핑 반짝임   | Pool In-Game 보드 내부, Title 전체 배경 |

정적 이미지는 **행성 스프라이트와 어울리는 캐주얼 게임풍 은하 배경**만 담당한다. 별·점·반짝임은 같은
이미지에 고정해 넣지 않고, 렌더 레이어에서 반복 재생한다. 따라서 배경 이미지를 교체해도 반짝임
밀도·속도·가독성은 코드에서 일관되게 조절된다.

## 2. 정적 이미지 기준

- 기준 리소스는 16:9 가로형 cover 마스터이며, 코드의 런타임 원본 크기는
  `game/src/assets.ts`의 `ASSET_SIZES.boardBackground`가 정한다. 9:16, 1:2 같은 세로 화면과
  Pool In-Game 보드 내부는 이 원본을 비율 유지 cover로 중앙 크롭해 사용한다.
- 물결 밴드는 기존 9:16 리소스보다 촘촘하게 배치해 세로 중앙 크롭에서도 패턴이 비어 보이지 않게 한다.
- 화면 중앙에는 행성 식별을 방해하지 않는 **큰 세로 물결형 은하 밴드**를 둔다.
- 행성 리소스와 같은 **플랫한 캐주얼 모바일 게임 디자인**을 따른다. 실사 성운, 사진 질감, 입자 노이즈,
  사실적인 먼지 구름은 쓰지 않는다.
- 색은 짙은 남색 계열을 중심으로 낮은 채도 하이라이트를 쓰고, 바깥 네이비/골드 보드 프레임과 충돌하지
  않게 전체 명도는 낮게 유지한다.
- 정적 이미지에는 십자 별표, 점, speckle, sparkle mark를 박지 않는다. 이 요소들은 §3의 런타임 효과로만
  표현한다.
- 정적 이미지는 교체 가능한 리소스 슬롯이며, 프롬프트·원본·후처리·최종 경로를
  `game/public/assets/prompts/` 아래에 기록한다.

## 3. 런타임 루핑 효과 기준

- 작은 별·점광·입자는 deterministic seed 기반으로 배치해 새로고침 간 밀도가 크게 흔들리지 않게 한다.
- 알파와 스케일을 사인/삼각파 계열으로 반복해 **끊김 없는 루핑 반짝임**을 만든다.
- Pool In-Game에서는 보드 inner line 안쪽 배경 영역에 clip/mask를 적용한다.
- Title에서는 같은 효과 시스템을 화면 전체 배경에 적용한다.
- 반짝임은 행성·HUD·버튼보다 뒤에 있고, 행성 식별과 입력 타겟 가독성을 해치지 않는 낮은 알파로 유지한다.
