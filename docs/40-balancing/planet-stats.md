# 행성 단계별 수치 (반지름 · 기본 점수)

`raw: §9`(반지름), `raw: §7`(점수). 사다리 순서는 [[../10-concept/planet-ladder]] 정본.

| 단계 | 행성 | 권장 반지름(px) | 기본 점수 | 비고 |
|---|---|---|---|---|
| 1 | 수성 | 18 | — | 합성 결과가 아님 → **점수 없음** (큐/초기 랙에서만 등장) |
| 2 | 화성 | 21 | 10 | 수성+수성 합성 시 부여 |
| 3 | 금성 | 24 | 30 | |
| 4 | 지구 | 28 | 70 | |
| 5 | 해왕성 | 32 | 150 | |
| 6 | 천왕성 | 37 | 320 | |
| 7 | 토성 | 43 | 700 | |
| 8 | 목성 | 50 | 1500 | |
| 9 | 태양 | 58 | 3200 | **최종 단계** — 태양+태양은 합성 없이 일반 물리 충돌만 처리 |

- **점수 부여 규칙:** 점수는 *합성으로 새 행성이 생성될 때* 그 결과 행성 등급의 기본
  점수를 준다. 수성은 합성 결과로 생성되지 않으므로 점수 행이 비어 있다(`—`).
- **반지름 스케일링:** 위 px 값은 9:16 기준 보드의 권장값. 보드 크기를 바꾸면 9개 값을
  **같은 비율로** 스케일링한다(`raw: §9`).
- 머지 점수 = 이 기본 점수 → [[combo-scoring]].
